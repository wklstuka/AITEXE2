# AI 写作助手落地页设计框架 (Apple 极简风格)

基于“零配置、单窗口、全模态输入”的产品理念，本设计框架对标 Apple 的极简美学（大留白、大排版、毛玻璃、微阴影、无边框感），并巧妙运用淡色底纹进行视觉区隔。

## 📁 一、 文件组织结构 (File Organization)

采用原生 HTML5 + Tailwind CSS + 原生 JavaScript 的轻量级架构，无需复杂的构建工具，方便直接部署。

```text
/ai-landing-page
├── index.html          # 主页面结构 (包含所有 Section)
├── styles.css          # 自定义样式 (用于 Tailwind 无法覆盖的极简动画或特定 Apple 风格毛玻璃特效)
├── script.js           # 交互逻辑 (FAQ 手风琴折叠、平滑滚动、导航栏吸顶、Modal 弹窗等)
└── /assets
    ├── /icons          # 极简线框图标 (SVG格式，用于功能卡片)
    └── /images         # 界面演示图 (如 Hero 区域的极简输入框示意图)
```

---

## 🎨 二、 视觉规范 (Visual Guidelines)

*   **字体 (Typography)**：
    *   主字体：`Inter`, `-apple-system`, `sans-serif` (干净、现代的无衬线体)。
    *   装饰字体：`Playfair Display`, `serif` (用于背景花体英文，增加高级感与人文气息)。
*   **色彩 (Color Palette)**：
    *   主背景色：`#fbfbfd` (Apple 经典的极浅灰白，比纯白更柔和)。
    *   文字主色：`#1d1d1f` (深灰黑，避免纯黑的刺眼)。
    *   辅助背景色：`#f5f5f7` (用于板块交替时的淡色底纹区隔)。
    *   强调色：纯黑 `#000000` (用于核心按钮和图标)。
*   **图标 (Icons)**：
    *   摒弃所有卡通 Emoji。
    *   全部采用极简的黑色线条 SVG 科技图标 (如 Heroicons)。
*   **交互 (Interactions)**：
    *   平滑的过渡动画 (Transition duration 300ms-500ms)。
    *   卡片悬停时的微缩放 (`scale-101`) 和微阴影 (`shadow-sm`)。
    *   弹窗 (Modal) 采用背景模糊 (`backdrop-blur`) 和平滑的缩放淡入效果。

## 📐 三、 页面结构划分 (Section 划分)

页面自上而下分为以下 7 个核心区块，通过背景色的交替 (`#fbfbfd` 与 `#ffffff` 或 `#f5f5f7`) 形成自然的视觉区隔。

### 1. Header (全局导航栏)
*   **视觉风格**：吸顶毛玻璃效果 (Backdrop-blur)，极简透明。
*   **内容**：
    *   左侧：产品 Logo (纯文字或极简几何图形)。
    *   右侧：锚点链接 (知识库、功能、定价、FAQ) + 突出的 CTA 按钮 ("立即体验")。

### 2. Hero (首屏核心区)
*   **背景**：淡色底纹 (`#fbfbfd` 或 `#f5f5f7`)。
*   **视觉风格**：居中对齐，超大无衬线字体，大面积留白。
*   **内容**：
    *   **大标题**：*"化繁为简，知识自然沉淀。"*
    *   **副标题**：*"无需繁琐设置。将文字、录音、图片或视频拖入同一窗口，AI 即可自动梳理核心知识与会议纪要。"*
    *   **核心交互演示**：页面中央展示一个极简的、类似 Spotlight 搜索框的 UI 组件，带有“拖拽文件至此”的微动画，完美体现“同一个窗口 (Single Window)”的产品理念。

### 3. Knowledge Base (我的知识库)
*   **背景**：纯白 (`#ffffff`)。
*   **视觉风格**：横向排列的 4 个极简卡片。卡片背景带有浅浅的 `Playfair Display` 花体英文水印 (Knowledge, Network, Meetings, Action)，提升高级感。
*   **内容**：
    *   **标题**：*"我的知识库"*
    *   **副标题**：*"一切，都在这一个窗口。强大的底层能力，包裹在最极简的交互之中。"*
    *   **卡片 1：知识图谱** (自动合并同类话题，构建结构化知识网络)
    *   **卡片 2：人脉网络** (基于音色与人脸识别，沉淀专属交互记忆)
    *   **卡片 3：会议纪要** (智能提取核心决议与风险点，告别冗长录音)
    *   **卡片 4：智能待办** (从对话中精准提取 Action Items，绝不遗漏)
*   **交互**：点击任意卡片，弹出居中的 Modal 弹窗，展示具体的子页面内容（如树状图节点、联系人列表、时间轴、复选框列表）。弹窗内部图标采用极简 SVG。

### 4. Features (功能展示区)
*   **背景**：淡色底纹 (`#f5f5f7`)。
*   **视觉风格**：紧凑精致的 2x2 网格卡片，纯白背景，极简黑色线条 SVG 图标。
*   **内容**：
    *   **标题**：*"重塑记录体验。"*
    *   **副标题**：*"打破格式壁垒，释放前所未有的生产力。"*
    *   **卡片 1：无界输入** (支持图文音视全格式，打破壁垒，随心记录)
    *   **卡片 2：无感运行** (告别繁琐表单。AI 自动识别意图，即刻响应)
    *   **卡片 3：智能重构** (碎片信息瞬间转化为结构化资产，井然有序)
    *   **卡片 4：秒级唤醒** (强大的语义检索，专属知识库随时待命)

### 5. Pricing (定价表)
*   **背景**：纯白 (`#ffffff`)。
*   **视觉风格**：极简的双栏对比卡片。
*   **内容**：
    *   **基础版 (Free)**：浅灰色卡片，满足日常轻度记录。
    *   **Pro 版 (Pro)**：纯黑色卡片，白字，突出“按年订阅”和“无限模态输入”。

### 6. FAQ (常见问题)
*   **背景**：淡色底纹 (`#fbfbfd` 或 `#f5f5f7`)。
*   **视觉风格**：极简手风琴 (Accordion) 折叠列表，纯白卡片，点击展开时带有平滑的过渡动画。
*   **内容 (预设 4 个问题)**：
    1.  支持哪些格式的录音和视频？
    2.  我的数据隐私如何得到保障？
    3.  是否需要学习复杂的提示词 (Prompt)？
    4.  如何导出整理好的知识点？

### 7. Footer (页脚)
*   **背景**：纯白 (`#ffffff`)。
*   **视觉风格**：干净的底部区域，浅灰色文字，顶部带有一条极细的分割线。
*   **内容**：版权信息、法律链接、社交媒体图标。
